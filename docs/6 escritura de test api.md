## Escritura de test de una API

### Descripcion
Validar que se permite obtener un producto mediante el parametro *id*

### Precondiciones 
- Colección importada en postman:
 6 saucedemo api.postman_collection.json

### Pasos de la prueba
1. Enviar la petición obtener access token
2. Enviar la petición Obtener producto mediante id
3. Validar que todos los tests hayan aprobado. 

### Resultado
Se obtiene una respuesta exitosa con los datos del producto.
