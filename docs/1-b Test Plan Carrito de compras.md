# Plan de pruebas de Carrito de compras

### objetivo:  
Validar el correcto funcionamiento del proceso de agregar, remover y visualizar items en el carrito de compras asegurando la correcta navegación y comportamiento esperado del sistema.

### Alcance de las pruebas:  
Se incluyen los escenarios positivos y negativos.  
No se incluyen pruebas del proceso para finalizar la compra.

### Entorno de Pruebas:
- Entorno: QA
- Url: [saucedemo](https://www.saucedemo.com)
